# Weekly Reports

This folder contains the weekly progress reports of the internship project conducted at the International Water Research Institute (IWRI), UM6P.

Each report documents:

- Work completed during the week  
- Methodological decisions and adjustments  
- Feedback received from supervisors  
- Implemented improvements  
- Next steps aligned with the Plan van Aanpak (PvA)  

The reports provide a structured overview of the project evolution from exploratory analysis toward spatial hydro-vegetation assessment and modelling preparation.
